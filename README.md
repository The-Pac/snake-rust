# Snake-Rust

Snake-Rust is a simple console-based test project to explore dynamic development in Rust. The goal is to create a basic implementation of the classic Snake game while getting familiar with Rustâ€™s syntax and capabilities for console applications.

## Features

- **Basic Snake Game Mechanics**: Play the traditional Snake game in your terminal.
- **Dynamic Console Interaction**: Learn how to handle dynamic console input and rendering.
- **Simple Game Loop**: Understand the structure of a game loop in Rust.
- **Basic Collision Detection**: Handle snake movement and detect collisions with the walls and itself.
